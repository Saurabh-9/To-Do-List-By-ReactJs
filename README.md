# Dynamic To-Do List with Drag-and-Drop and Dark Mode  

An interactive React-based to-do list application with features like drag-and-drop task reordering, deadline management, progress tracking, and customizable dark mode.  

## Features  
- **Add New Tasks**: Create tasks with names and deadlines.  
- **Task Management**: Mark tasks as completed, delete, or pin them as favorites.  
- **Drag-and-Drop Reordering**: Prioritize tasks dynamically using drag-and-drop.  
- **Progress Tracking**: View progress with a percentage of completed tasks.  
- **Dark/Light Mode Toggle**: Switch between themes for a personalized user experience.  

## Technologies Used  
- **React**: Component-based architecture for a dynamic user interface.  
- **CSS**: Styling for light/dark themes and task elements.  
- **react-beautiful-dnd**: Implementing drag-and-drop functionality.
